Real Estate API Documentation

Welcome to the Real Estate API documentation. This API provides endpoints for managing properties, users, and more.

Authentication

POST /api/auth/token/

Get authentication token

Request Body:
{
    "email": "user@example.com",
    "password": "your_password"
}
Response:
{
    "access": "access_token",
    "refresh": "refresh_token"
}

Properties

GET /api/properties/

Get list of properties with filtering options

Query Parameters:
{
    "location": "state_id",
    "price_min": "number",
    "price_max": "number",
    "property_type": "string",
    "bedrooms": "number",
    "page": "number"
}
POST /api/properties/

Create new property listing

Request Body:
{
    "title": "string",
    "description": "string",
    "price": "number",
    "location": "number",
    "property_type": "string",
    "bedrooms": "number",
    "bathrooms": "number",
    "area": "number",
    "images": "array"
}

Users

POST /api/users/register/

Register new user

Request Body:
{
    "email": "string",
    "password": "string",
    "first_name": "string",
    "last_name": "string",
    "phone": "string"
}

Locations

GET /api/locations/states/

Get list of Algeria states

GET /api/locations/cities/{state_id}/

Get cities for a specific state

Likes & Views

POST /api/properties/{id}/like/

Like/Unlike a property

GET /api/properties/{id}/view/

Record a view for a property
